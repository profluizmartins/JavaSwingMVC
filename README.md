# Atividade da Semana

A atividade desta semana consiste na evolução do CRUD de produtos desenvolvido anteriormente. Nesta etapa, você deverá:

- Implementar validações utilizando **tratamento de exceções**;
- Documentar as classes com **Javadoc**;
- Realizar a entrega **exclusivamente via commit no Git**  
  *(desconsidere qualquer envio pelo SIGAA)*.

## Instruções para Entrega

1. Acesse o repositório do projeto:
   - https://github.com/profluizmartins/JavaSwingMVC

2. Crie uma **branch com o seu nome** no repositório.

3. Submeta o seu código **somente dentro dessa branch**.
